## 📋 Project Overview
This project is a deep-dive into the **NHANES (National Health and Nutrition Examination Survey) 2020 dataset**. Using high-performance numerical computing with **NumPy**, I analyzed physical measurements from adult males and females to identify trends in weight, body composition, and metabolic health indicators.

## 🔍 Scientific Questions Explored
* **Distribution:** How does weight vary across the population? (Spoiler: It's heavily right-skewed!)
* **Metrics:** Is BMI enough, or do Waist-to-Height and Waist-to-Hip ratios tell a deeper story?
* **Correlation:** How closely linked are height, weight, and waist circumference in a standardized population?

## 🛠️ The Data Science Toolkit
* **Engine:** `NumPy` for multidimensional matrix operations.
* **Visualization:** `Matplotlib` & `Seaborn` for statistical plotting.
* **Mathematics:** `SciPy` for skewness and correlation coefficients.
* **Environment:** Google Colab / Jupyter Notebooks.

## 📊 Key Insights from the Lab
| Analysis | What I Found |
| :--- | :--- |
| **Weight Symmetry** | Both genders show a significant "right tail," meaning a few high-weight outliers pull the average up. |
| **The BMI Gap** | While BMI is useful, the **Waist-to-Height ratio** proved to be a more focused indicator of central obesity. |
| **Variable Links** | There is a massive correlation between Waist Circumference and BMI, far outweighing the link between Height and BMI. |
